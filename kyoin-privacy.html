AI授業メーカー プライバシーポリシー

合同会社Shisto

合同会社Shisto(以下「当社」といいます)は、当社が提供する授業計画生成サービス「AI授業メーカー」(以下「本サービス」といいます)における、利用者の個人情報の取り扱いについて、以下のとおりプライバシーポリシー(以下「本ポリシー」といいます)を定めます。

第1条(取得する情報)

当社は、本サービスの提供にあたり、以下の情報を取得します。

  1. 利用者から直接ご提供いただく情報
    • 氏名
    • メールアドレス
    • 所属(学校名等)
    • 領収書の宛名
    • 振込名義
    • その他、お問い合わせ等の際に利用者が任意にご提供いただく情報
  2. 利用者が本サービスに入力する情報
    • 授業計画の生成にあたって入力される学年、教科、単元等の情報
  3. 本サービスの利用により自動的に取得される情報
    • アクセスログ(アクセス日時、IPアドレス、ブラウザ種別等)
    • 本サービスの利用履歴

第2条(利用目的)

当社は、取得した情報を以下の目的で利用します。

  1. 本サービスの提供および運営のため
  2. 利用者からのお問い合わせに対応するため
  3. 利用契約の締結、利用料金の請求、領収書の発行のため
  4. 本サービスの改善および新機能の開発のため
  5. 利用規約違反への対応のため
  6. 本サービスに関する重要なお知らせを送付するため
  7. 本サービスの利用状況に関する統計データの作成のため(個人を識別できない形式に加工した上で利用します)

第3条(第三者提供)

  1. 当社は、利用者の同意を得ることなく、個人情報を第三者に提供することはありません。ただし、次の各号に該当する場合はこの限りではありません。
    1. 法令に基づく場合
    2. 人の生命、身体または財産の保護のために必要がある場合であって、本人の同意を得ることが困難であるとき
    3. 公衆衛生の向上または児童の健全な育成の推進のために特に必要がある場合であって、本人の同意を得ることが困難であるとき
    4. 国の機関もしくは地方公共団体またはその委託を受けた者が法令の定める事務を遂行することに対して協力する必要がある場合
  2. 前項にかかわらず、次条に定める外部サービスへの情報提供は、本サービスの提供に不可欠なものとして、利用者の同意の上で行います。

第4条(外部サービスへの情報提供)

  1. 本サービスは、授業計画の生成にあたり、Google社(Google LLC)が提供する生成AIサービス「Gemini API」を利用しています。
  2. 利用者が本サービスに入力した情報(学年、教科、単元等)は、授業計画の生成処理のため、Gemini APIに送信されます。
  3. Gemini APIにおけるデータの取り扱いについては、Google社のプライバシーポリシーおよびGemini APIの利用規約に準拠します。詳細は以下をご参照ください。
    • Google プライバシーポリシー：https://policies.google.com/privacy
    • Gemini API 利用規約：https://ai.google.dev/gemini-api/terms
  4. 利用者は、本サービスを利用することにより、前各項の情報提供に同意したものとみなします。
  5. 利用者は、機微な個人情報(児童・生徒の個人情報、機密情報等)を本サービスに入力しないよう十分にご注意ください。入力された情報についての責任は利用者が負うものとし、当社は一切の責任を負いません。

第5条(情報の管理)

  1. 当社は、取得した個人情報について、漏えい、滅失、毀損の防止その他の安全管理のために必要かつ適切な措置を講じます。
  2. 当社は、個人情報の取り扱いを委託する場合には、委託先において適切な安全管理措置が講じられるよう、必要な監督を行います。
  3. 本サービスの提供にあたって利用する本サービスの稼働基盤(Render等のクラウドホスティングサービス)は、いずれも一般的なセキュリティ水準を満たす事業者のサービスを利用しています。

第6条(情報の保存期間)

  1. 当社は、取得した個人情報を、利用目的の達成に必要な期間に限り保存します。
  2. 利用契約終了後、一定期間(原則として終了から1年間)経過後に、法令等で保存が義務付けられている情報を除き、個人情報を削除します。
  3. アクセスログ等の自動取得情報は、取得から最長12ヶ月保存します。

第7条(Cookie等の利用)

  1. 本サービスは、利用者の利便性向上およびセッション管理のため、Cookieを使用する場合があります。
  2. 利用者は、ブラウザの設定によりCookieの受け入れを拒否することができます。ただし、Cookieを無効にした場合、本サービスの一部機能が利用できなくなる可能性があります。

第8条(個人情報の開示・訂正・削除)

  1. 利用者は、当社に対し、自己の個人情報の開示、訂正、追加、削除、利用停止、消去、第三者への提供の停止(以下「開示等」といいます)を請求することができます。
  2. 開示等の請求は、第11条に定めるお問い合わせ窓口までご連絡ください。
  3. 当社は、利用者本人であることを確認の上、法令の定めに従い、速やかに対応いたします。
  4. 個人情報の削除請求にお応えした場合、本サービスの一部または全部を利用できなくなる場合がありますのでご了承ください。

第9条(未成年者の個人情報)

本サービスは、小学校の教員を主たる対象としており、未成年者の利用を想定していません。万一、未成年者の個人情報を取得してしまった場合、速やかに削除します。

第10条(プライバシーポリシーの変更)

  1. 当社は、必要に応じて本ポリシーを変更することがあります。
  2. 本ポリシーを変更する場合、変更後の本ポリシーをウェブサイトに掲示した時点から効力を生じるものとします。
  3. 重要な変更がある場合は、利用者に対し電子メール等の方法で通知します。
